# ST-027 — Organisation-wide availability and bench view

**Story ID:** ST-027
**Epic:** EP-05 — Team Assignment & Availability
**Priority:** P2
**Points:** 5
**Status:** Not Started

---

## User Story

As a CTO/CEO or PM, I need an organisation-wide view of every employee's availability status, so that I can quickly identify who is free or partially free when a new project needs staffing.

---

## Acceptance Criteria

- [ ] Availability view lists all active employees
- [ ] Each row shows: name, role, availability status, current project (if assigned)
- [ ] Availability status values: Available (no active assignment), Assigned (active assignment, not partially available), Partially Available (active assignment + flag set)
- [ ] Employees with Available status appear at the top — then Partially Available — then Assigned
- [ ] View is accessible to CTO/CEO (all employees) and PM (all employees — read only, no assignment action)
- [ ] View updates in real time when assignments are made or removed

---

## Notes

- This view complements the People Directory (EP-02) — directory is about skills, availability view is about capacity
