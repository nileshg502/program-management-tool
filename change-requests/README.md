# Change Requests

## Overview

Formal change requests for scope changes after requirements have been baselined.

## Process

1. Copy [CR-TEMPLATE.md](CR-TEMPLATE.md) to `active/CR-NNN-title.md`
2. Fill in all sections
3. Route for approval per the Approval section
4. Move to `change-request-log.md` once resolved

## Status Values

- **Active** — Under review
- **Approved** — Approved, ready for implementation
- **Rejected** — Not approved, reason documented
